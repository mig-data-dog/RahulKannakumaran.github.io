Meeting Minutes:

We made and used a discord group to communicate with each other throughout the duration of this project. In the discord we:

Discussed when we would meetup to work on our presentation and porfolio.
Added all of the notes which we found during our research.
Traded all of the code we wanted to add to the website (to promote code management).
Made a list of every resource which we used when researching.
Found all of the photos we wanted to use in the website and included their source when posting them.

We meetup as a group after every COMP501 lecture as we always had a break after this. During these meetings some of the things we discussed were:

  • What material we should add to our slide
  • What resources we should use
  • What role is best for what team member
  • What we want the website to look like
  • What risks we want to focus on and how those risks impact Machine Learning

We also created a google docs where we would write all of the information we wanted to add to the website.

We used this google doc as the main way to give each other the information. And the final material which is added to the website is copied and pasted from this google docs.

This doc has made communication each others work so much easier. And allows for each team member to read over and fix any grammar/punctuation issues which may not have been found when it was intially written.
